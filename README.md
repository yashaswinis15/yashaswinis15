## 🚀 Featured Projects

### 🔴 FinStream — Real-Time Fraud Detection & Analytics Pipeline
> A production-style streaming pipeline that ingests financial transactions via CDC, processes them through a Medallion architecture, and surfaces fraud signals in near real-time.

**Architecture:** `PostgreSQL` → `Debezium (WAL-based CDC)` → `Kafka` → `Spark Structured Streaming` → `Delta Lake (Bronze/Silver/Gold)` → `dbt` → `Grafana + Slack Alerts`

**Highlights:**
- ✅ At-least-once CDC delivery with idempotent Spark processing → effectively-once behavior
- ✅ Fault-tolerant: Kafka offset management + Spark checkpointing + Delta Lake transaction log
- ✅ Schema evolution support with minimal pipeline disruption
- ✅ Gold layer business aggregations: fraud summaries, volume trends, account risk profiles
- ✅ Alert engine polling PostgreSQL every 30s for high-risk activity
